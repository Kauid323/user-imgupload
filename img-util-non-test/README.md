# img-util-non-test

This folder contains refactors of `img-util-python` into various languages from the provided list.

Notes:

- Some entries are **general-purpose languages** (e.g. Ada, AppleScript) and include an end-to-end implementation.
- Some entries are **DSL / config / grammar formats** (e.g. ABNF, ApacheConf, Adblock Filter List, ANTLR, Ant build system). For those, the files describe configuration or protocol/grammar representations of the same workflow.

Each subfolder has its own minimal README/usage notes inside.
